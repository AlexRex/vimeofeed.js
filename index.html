Vimeo Feed JS

Implement a feed from your vimeo videos.

USAGE

  1. Put the script from the archive vimeofeed.js at HEAD.
  2. Change the ID and if you want the Name of the ID div where the videos will be.
  3. Modify the for loop to get more or less videos.
  4. Have fun.

You can download all the content to see a demo page or visit http://www.alexrex.com/vimeofeed/.
